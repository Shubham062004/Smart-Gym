# Smart-Gym

Smart-Gym is an AI-powered fitness training application that turns your camera into a personal trainer.

Using real-time pose detection and movement analysis, Smart-Gym can automatically:

• Count your workout repetitions
• Track your exercise performance
• Analyze posture and form
• Provide instant feedback

All of this works using only your phone camera — no wearables required.

## Features

* AI Pose Detection
* Automatic Rep Counting
* Real-time Posture Feedback
* Workout Tracking
* Performance Analytics
* Privacy-first processing (camera processed locally)
